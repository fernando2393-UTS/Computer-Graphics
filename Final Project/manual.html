  1. Main Features
    1. Change from a first person camera (Moving Camera) to an orbit camera (Orbit Camera)
    2. Move around the house using the keys W,A,S,D, press ESC to get control over the mouse again, press SPACE to switch back to the first person movement without mouse pointer
    3. Move objects by clicking on them and placing them to a free space (collision detection available)
    4. Only specific objects are selectable (e.g. no walls can be selected)
    5. Deselect items by pressing ESC
    6. Change texture of the floor and the outer wall
    7. Adjust individual lighting for all lamps and the camera light as well

  2. Programming Insights
    1. Code is structured in separate JS files
    2. Index file does only contain the project basics like creating the scene and setting up the renderer
    3. Main procedure is done by calling functions from other scripts and therefore creating the house and its objects
    4. Models used are in .ply-format and can be adjusted with MeshLab
    5. Bounding Boxes are used for collision detection and an alert is displayed in that case
    6. Texture loading is done with customized methods which also allows for updating specific sides of a mesh
    7. A css-file in 'css'-folder is used to style the HTML elements
    8. All textures can be found inside the 'img'-folder
    9. All models can be found inside the 'models'-folder

  3. Scripts Overview
    dat.gui.min.js Third party script to build the GUI
    gui.js Custom script to build the GUI using dat.gui.min.js library
    house.js Custom script to create the walls, floor, windows, doors, lighting, sphere, and ground
    keyControls.js Custom script defining all key controls like W,A,S,D movement or ESC function
    meshOps.js Custom script performing all texture creating and updating
    modeladder.js Custom script adding and adjusting all objects/models to the correct position
    movingCamera.js Custom script creating a moving camera
    orbitCamera.js Custom script creating an orbit camera
    OrbitControls.js Third party script managing the controls for the orbit camera
    PLYLoader.js Third party script allowing to load .ply-models and adding them to a three.js-scene
    PointerLockControls.js Third party script managing the controls for the moving camera
    selector.js Custom script containing the engine for selecting and placing specific objects including collision detection
    three.js / three.min.js / three.module.js Third party scripts - core libaries

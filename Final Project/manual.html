Manual Page Final Computer Graphics Project

Contributors: Alexander Ebner, Fernando Garcia, Hendrik Hangen, Marlies Wagner


  1. Main Features
    1. Change from a first person camera (Moving Camera) to an orbit camera (Orbit Camera)
    2. Move around the house using the keys W,A,S,D
    3. Move objects by clicking on them and placing them to a free space (collision detection available)
    4. Deselect items by pressing ESC
    5. Change texture of the floor and the outer wall
    6. Adjust individual lighting for all lamps and the camera light as well

  2. Programming Insights
    1. Code is structured in separate JS files
    2. Index file does only contain the project basics like creating the scene and setting up the renderer
    3. Main procedure is done by calling functions from other scripts and therefore creating the house and its objects
    4. Models used are in .ply-format and can be adjusted with MeshLab
    5. Bounding Boxes are used for collision detection and an alert is displayed in that case
    6. Texture loading is done with customized methods which also allows for updating specific sides of a mesh
